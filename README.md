# Golden perch recruitment
 Code to accompany:
 Schilling, H., A.Brooks, K.Cheshire, et al. 2025. “Multi-Scale Influences of Climate, River Hydrology and Temperature, and Management Activities on Recruitment of a Periodic Fish Species (Golden Perch Macquaria ambigua).” Ecohydrology18, no. 5: e70095. https://doi.org/10.1002/eco.70095.

 Due to government policy, data can only be shared upon completion of a data sharing licence with the NSW Department of Primary Industries Fisheries.
